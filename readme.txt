عنوان پروژه:
ماشین حساب ماتریس و بردار

پیاده‌سازی شده توسط:
مهتاب کمانی، به شماره دانشجویی 1404010121004

فایل‌های پروژه:
1. matrix_library.h => توابع، متغیرها، کتابخانه‌ها
2. matrix_library.cpp => اجرای برنامه
3. demonstration.cpp => تابع اصلی
4. test_cases.txt => تست تمامی قابلیت‌ها
5. README.txt => توضیحات تکمیلی

ویژگی‌ها:
1. عملیات تک ماتریسی: ضرب اسکالر، ترانهاده، نمایش قطر اصلی و فرعی، محاسبه دترمینان به روش بازگشتی، محاسبه معکوس ماتریس
2. عملیات دو ماتریسی: جمع، تفریق، ضرب
3. عملیات تک برداری: ضرب اسکالر، پیدا کردن اندازه، نرمال‌سازی
4. عملیات دو برداری: جمع، تفریق، ضرب داخلی، ضرب خارجی
5. خطایابی دقیق با توضیحات تکمیلی برای دلیل وقوع خطا و مقصد بعدی
6. منوی دقیق با قابلیت بازگشت

محدودیت‌ها:
1. ماتریس‌ها حداکثر 1000×1000
2. طول بردار حداکثر 1000

منابع و مراجع:
1. جزوات و سرفصل‌های دانشگاه، درس مبانی برنامه‌سازی
2. دوره آموزشی برنامه نویسی به زبان سی پلاس پلاس دکتر فرشید شیرافکن در فرادرس
3. راهنمایی‌های تخصصی چند تن از دوستان آشنا به منطق برنامه‌سازی => بهینه‌سازی
4. هوش مصنوعی دیپ‌سیک و چت جی‌پی‌تی => ویرایش و گرفتن خطا
